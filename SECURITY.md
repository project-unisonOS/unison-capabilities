# Security

This repository contains the capability registry, resolver, and onboarding flow for UnisonOS.

For overall security architecture and platform hardening guidance, see:

- https://project-unisonos.github.io/reference/security/

If you believe you have found a security issue in capability discovery, installation, credential handling, or execution policy enforcement, please follow the responsible disclosure guidance in the UnisonOS security docs or open a private security advisory on GitHub instead of filing a public issue.
